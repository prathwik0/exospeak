Integrate in Minutes

Adding Chotu is incredibly simple. Just copy the script below and paste it before the closing `</body>` tag in your HTML. 


<script type="module">
    import React from "https://esm.sh/react@latest"
    import ReactDOMClient from "https://esm.sh/react-dom@latest/client"
    import { Assistant } from 'https://esm.sh/lp-assistant@0.1.11-b'

    window.addEventListener('load', () => {
        const container = document.createElement('div');
        container.id = 'ai-assistant-container';
        document.body.appendChild(container);
        ReactDOMClient.createRoot(container).render(
            React.createElement(Assistant, {
                apiUrl: 'custom_api_url', // Replace with your API URL
                modelName: 'model' // Replace with your model name
            })
        );
    });
</script>

The Chotu assistant is already running on this page! Look for the chat icon in the bottom-right corner.
